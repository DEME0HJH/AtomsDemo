# 任务划分及原因

## 实施路线图

### Phase 1：模板系统扩展（1-2 天）
建立 5 分类 16 个参数化模板 + 智能关键词匹配

### Phase 2：真正的多智能体（2-3 天）
5 个 Agent 各自调用 LLM + Pipeline 编排 + Zod 校验 + Agent 文档生成

### Phase 3：Monaco Editor 升级（1 天）
@monaco-editor/react 替换 textarea

### Phase 4：后端与云端存储（3-4 天）
Supabase 集成（Auth + PostgreSQL + 同步）+ 优雅降级

### Phase 5：部署与分发（2-3 天）
移除 static export → Vercel 真实部署 → 分享页面

### Phase 6：安全与测试（2-3 天）
AES-256-GCM 加密、iframe sandbox、JSON 4 层回退解析、测试补完

## 依赖关系

```
Phase 1 ──→ Phase 2 ──→ Phase 3
Phase 4 ──→ Phase 5 ──→ Phase 6
```
