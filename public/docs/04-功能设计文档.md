# 功能设计文档

## 模块架构

### ChatInterface
用户输入、Agent 消息流展示、16 模板分类选择器

### AppPreview
4 Tab 模式：预览(iframe+sandbox) | 源码 | Monaco编辑器 | 对话记录

### AgentEngine + Pipeline
核心编排：有 Key → 5 Agent 真实 LLM 协作，无 Key → 仿真+模板回退

### Agent 系统（src/lib/agents/）
- **BaseAgent**：LLM 调用、重试、4 层 JSON 解析回退
- **Pipeline**：4 种工作流编排 + 文档生成
- **Docs**：各 Agent 输出 → Markdown 格式化
- **Registry**：Agent 注册与发现

### 存储
- 本地：localStorage（主）+ IndexedDB（备）+ AES-256-GCM 加密 Key
- 云端（可选）：Supabase Auth + PostgreSQL + 双向同步

## 数据模型

```typescript
interface GeneratedApp {
  id: string;
  name: string;
  type: string;
  code: { html: string; css: string; js: string; files?: AppFile[] };

  path: string;     // "index.html" / "docs/01-需求分析.md"
  content: string;
```
