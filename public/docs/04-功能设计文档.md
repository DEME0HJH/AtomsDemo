# 功能设计文档

## 组件架构

### 1. AgentTeam
- **功能**：展示 7 个 AI Agent 的头像和角色
- **交互**：hover 效果，点击可查看详情（扩展）
- **Props**：无

### 2. ChatInterface
- **功能**：用户输入、Agent 消息展示、生成步骤可视化
- **状态**：input, messages, steps, isGenerating
- **交互**：发送消息、选择模板、查看生成进度

### 3. AppPreview
- **功能**：预览生成的应用、查看源代码、下载 HTML
- **状态**：activeTab ('preview' | 'code')
- **交互**：iframe 预览、代码高亮、文件下载

### 4. HistoryPanel
- **功能**：展示生成历史列表，支持选择和删除
- **状态**：apps, isOpen
- **交互**：展开/收起、选择应用、删除记录

### 5. ModeSelector
- **功能**：切换工程师模式和团队模式
- **Props**：mode, onChange

## 数据流

```
User Input → ChatInterface → AgentEngine → Template Matching
                                      ↓
                              GeneratedApp → localStorage
                                      ↓
                              AppPreview (iframe render)
```

## 状态管理

采用 React 原生状态管理（useState + useCallback）：
- 主页面管理 mode、selectedApp、showPreview
- ChatInterface 独立管理消息和生成状态
- HistoryPanel 通过 key 强制刷新

## 数据模型

### GeneratedApp
```typescript
interface GeneratedApp {
  id: string;
  name: string;
  description: string;
  type: 'todo' | 'weather' | 'calculator' | 'custom';
  code: { html: string; css: string; js: string };
  createdAt: number;
  prompt: string;
  steps: GenerationStep[];
}
```
