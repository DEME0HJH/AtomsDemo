# 实现思路与关键取舍

## 整体架构

Atoms Demo 基于 Next.js 14 + React 18 + TypeScript + Tailwind CSS，四层架构：

1. **表现层**：React 组件 + Monaco Editor + Tailwind 暗色主题
2. **业务逻辑层**：5 个真实 AI Agent + Pipeline 编排 + Zod 类型校验
3. **服务层**：Next.js API Routes（DeepSeek 代理）+ Supabase（可选云存储）
4. **存储层**：localStorage + IndexedDB（本地）+ Supabase PostgreSQL（可选）

## 核心取舍

| 决策 | 选择 | 原因 |
|------|------|------|
| AI 模式 | 5 Agent 独立 LLM 调用 | 真正的知识传递，而非假模拟 |
| 无 API Key 时 | 16 个参数化模板回退 | 确保任何用户都能生成应用 |
| 部署 | 移除 static export | API Routes 需要在服务端运行 |
| 存储 | 本地优先 + 可选 Supabase | 未配置时优雅降级，核心功能不受影响 |
| 编辑器 | Monaco Editor | 专业体验，代价是 ~2MB 包体积 |
| API Key | AES-256-GCM 加密 | 设备绑定密钥，不可跨设备解密 |
| JSON 解析 | 4 层回退策略 | 解决 LLM 返回代码中未转义字符问题 |
