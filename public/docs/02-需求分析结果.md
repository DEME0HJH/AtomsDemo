# 需求分析结果

## 核心功能

### AI 多智能体协作（P0）
5 个 AI Agent 各自调用 LLM，形成知识传递链：Mike → Emma → Bob → Alex → David。支持 team/engineer/review/plan 四种工作流。

### 应用代码生成（P0）
根据用户描述生成完整 HTML/CSS/JS 单页应用。AI 模式下附带 5 份 Agent 分析文档。

### 模板系统（P0）
16 个参数化模板覆盖 5 个分类：生产力、工具、创意、数据、游戏。无 API Key 也能使用。

### 代码编辑器（P1）
Monaco Editor 替代纯 textarea，支持语法高亮、代码补全、格式化、多文件 Tab。

### 数据管理（P1）
localStorage + IndexedDB本地存储，可选 Supabase 云端同步，API Key AES-256-GCM 加密。
